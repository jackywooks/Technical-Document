CSS

CSS stands for Cascading Style Sheets. It is a language that controls how HTML elements are displayed on web pages. It can change the appearance, layout, and animation of web elements. It can also apply different styles for different devices and screen sizes. CSS files are external to HTML files

p {
color: red;
text-align: center;
}

Javascript

JavaScript is a language that enables dynamic and interactive web pages. It can manipulate HTML, CSS, and other web elements, as well as create functions, variables, and objects. It can also handle events, data, and logic. It runs on web browsers and can communicate with servers and databases

<button type="button" onclick='document.getElementById("demo").innerHTML = "Hello JavaScript!"'>Click Me!</button>

PHP

PHP is a language that enables server-side web development. It can embed HTML, CSS, and JavaScript code, as well as interact with databases and files. It can also handle forms, sessions, cookies, and other web features. It is widely used and supported by many web frameworks and platforms
